This game is about a hero from the main game series Ghumn.

I did all the programming and game design.

### Some cool stuff done
* Used singleton pattern for some managers (game manager, UI manager and etc)
* Created some object pooling for the enemies and bullets
* Used Audio Mixer for the audio stuff
* Used the new input system
* Created a wave system wich controls every base enemy, so I can do some actions in one update and save some calls
* Created multiple boss mechanics
* Used composition with the bosses, dividing everything in pieces that have their own role

### Technology
- Unity3D
- Rider
- Aseprite
- Bfxr (for the sound effects)
